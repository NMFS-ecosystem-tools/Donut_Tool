class  nmfDonutModelAPI
 This is the API class for the Rank Proportion Algorithm. All of the calculations that go into generating the Proportion and Output tables are performed by this class. There are no GUI calls in this class. It simply accepts a matrix, operates on it, and then sends it back to the calling program (by reference). More...
